# Cyrillic Phonetic Layout for macOS

This directory contains the macOS bundle version of the Cyrillic Phonetic keyboard layout.

## Installation

1. Unzip the archive:
   ```sh
   unzip layout.bundle.zip
   ```

2. Move the resulting `.bundle` file into:
   ```
   ~/Library/Keyboard Layouts
   ```

3. Log out and log back in.

4. Add the layout in **System Settings → Keyboard → Input Sources**.
